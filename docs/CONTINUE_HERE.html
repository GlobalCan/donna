4Troubleshooting quick-reference

PowerShell won't run the script:
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
Bot connects but DMs do nothing:
MESSAGE CONTENT intent not enabled in Discord Developer Portal. Go to your bot's page → Bot → Privileged Gateway Intents → enable it → Save Changes → restart donna.main.
401 Unauthorized from Anthropic:
Billing not enabled on the Anthropic account. Add a card at console.anthropic.com/settings/billing.
The docs the new session needs aren't there:
Run git pull inside the repo — commits may have landed since cloning.
Totally stuck:
Paste the error into the old Claude Code session on the first laptop. It has the full context.
